How It Works (v5)
Detection pipeline

Each frame flows through this chain:

  • 1· One Euro Filter - adaptive low-pass smoothing on every face + hand landmark
  • 2· Multi-Zone Face Detection - 13 anatomical regions, centroid-based
  • 3· Hand Pose Classifier - pinch_grip / fist / pointing / open_hand / curled
  • 4· 4-state Rule Engine - IDLE → PROXIMITY → DWELL → BFRB with multi-signal scoring
  • 5· ML Classifier - 16-feature Random Forest (trained on Kaggle CMI sensor data), 1.5s window
  • 6· Late Fusion - fused_score = 0.6 · ML + 0.4 · rule
  • 7· Adaptive Threshold - per-session calibration sets the proximity threshold
  • 8· WebSocket Transport - real-time predictions over a persistent socket
  • 9· IEEE Telemetry - per-frame structured logs, batched to /api/telemetry
Improvements (Tier 1 + 2)
