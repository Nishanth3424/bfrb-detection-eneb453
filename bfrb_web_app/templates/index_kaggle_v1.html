How It Works

View explanation

This system bridges two data sources:

  • Training: Kaggle CMI wrist sensor data (acc/gyro/ToF/thermal) with labeled BFRB gestures.
  • Inference: MediaPipe landmarks extracted from your webcam in real-time.

Both are transformed into the same 16 semantic features:

  • Movement intensity ← acceleration / wrist velocity
  • Rotation speed ← quaternion / palm normal change
  • Skin contact ← thermal sensors / hand-face overlap
  • Proximity ← ToF depth grid / landmark distances

A Random Forest classifier trained on Kaggle sensor patterns detects the same patterns in MediaPipe landmark data.
